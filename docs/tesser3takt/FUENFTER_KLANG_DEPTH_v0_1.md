## 2. Four-Layer Depth Stack

[MODEL] The depth stack has four traversal layers.

Essence:
Unavailable origin, potential, pre-form, not owned or possessed.

Meaning:
Resonance, archetypal pattern, sense-making, symbolic relation.

Structure:
Organization, rule, code, architecture, repeatable relation.

Manifestation:
Visible form, interface, artifact, file, action, experience.

These layers are not ranked as better or worse. They are traversal layers. A movement may begin in any layer, but it must name the layer it occupies and the layer it attempts to reach.

## 3. Descent and Ascent

Descent:
Manifestation -> Structure -> Meaning -> Essence

Descent means moving from visible form toward underlying organization, resonance, and unavailable origin.

Ascent:
Essence -> Meaning -> Structure -> Manifestation

Ascent means moving from potential toward sense, organization, and concrete artifact.

Guard: Do not treat descent as escape from reality or ascent as guaranteed realization.

## 4. Movement Grammar

[MODEL] The Fuenfter Klang depth module uses three movement directions.

Vertical:
Descent or ascent between depth and form.

Horizontal:
Integration between registers, perspectives, or neighboring contexts.

Diagonal:
Transformation jump, Roesselsprung, context shift, non-linear transition.

Vertical movement gives depth. Horizontal movement gives integration. Diagonal movement gives transformation.

Guard: Diagonal movement must still pass guard and reentry. It is not a license for arbitrary jumps.

## 5. Integration Loop

[MODEL] The integration loop is:

Experience -> Understanding -> Integration -> Transformation -> Resonance -> Experience

Experience becomes readable. Understanding names a relation. Integration binds it into structure. Transformation changes the available path. Resonance returns the transformed state into lived experience.

Guard: The loop is a process model, not a psychological law.
