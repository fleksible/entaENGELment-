# Form in Bewegung — Learning Edge (Grammatik/Algebra als Sammeltechnik)

> Refrain: **Schichtung speichert. Erosion liest. Kante wird Oberfläche.**

## Kurzgeschichte (als technische Notiz, nicht als Mythos)
Ich hatte im Deutschunterricht lange solide Noten – bis Grammatik kam.
Ähnlich in Mathe: ab einem bestimmten Formalitätsgrad (z.B. binomische Formeln) brach es ein.
Gleichzeitig war Physik stark. Das Muster ist simpel:
**Intuition für Prozesse / Felder war da (1), Ausdruck war oft da (2), aber die Bindung an formale Regeln war schwach (5).**

Heute wird klar: **Genau diese "5er-Zone" ist nötig**, damit Erkenntnisse über Emergenz nicht nur erlebt, sondern **gesammelt, geprüft und navigierbar** werden.

## Was damals "5" war, ist heute die Klammer
- **Grammatik**: bindet Bedeutung an Struktur → macht Sprache reproduzierbar.
- **Algebra/Formeln**: binden Intuition an Transformationsregeln → machen Modelle überprüfbar.
- **Ohne Bindung** bleibt Erkenntnis ein Ereignis; **mit Bindung** wird sie ein Baustein.

## Repo-Übersetzung (warum das hier ins Projekt gehört)
Dieses Repo arbeitet bereits mit genau dieser Idee:
- **Claim-Tags / Receipts**: Aussagen werden nicht nur gesagt, sondern klassifiziert und rückverfolgbar gemacht.
- **Gate / Leakage**: Manifestation braucht Bedingungen; Bedeutung braucht Grenzen.
- **Navigation über Pointer/Index**: Wissen wird nicht "gefühlt", sondern auffindbar.

Diese Notiz ist die persönliche Begründung dafür, warum wir Form-Schichten brauchen:
**Damit Emergenz nicht verdampft, sondern versioniert werden kann.**

## Praktische Regel (für zukünftiges Schreiben im Repo)
Wenn eine Einsicht auftaucht, wird sie in drei Schichten abgelegt:
1) **RAW**: Quelle / Gespräch / Export (unverändert)
2) **EXTRACT**: markierte Passagen + Claim-Tags + Trigger ("wenn X, dann Y")
3) **SYNTH**: begehbare Treppe (Narration / Interface), die auf RAW/EXTRACT zeigt

## Claim-Tags (minimaler Lesecode)
- `[FACT]` beobachtbar / belegbar
- `[HYP]` Hypothese / Modellannahme
- `[MET]` Metapher / Interface-Sprache
- `[RISK]` Leakage / Verwechslungspotenzial
