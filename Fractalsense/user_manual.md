## Installation

### Android-Installation

1. **Voraussetzungen**
   - Android-Gerät mit Version 8.0 oder höher
   - Mindestens 2 GB freier Speicherplatz
   - Aktive Internetverbindung für die Ersteinrichtung

2. **Installation der APK**
   - Laden Sie die Datei "FractalSense_EntaENGELment.apk" auf Ihr Gerät
   - Öffnen Sie die Datei-App und navigieren Sie zum Download-Ordner
   - Tippen Sie auf die APK-Datei
   - Falls erforderlich, aktivieren Sie "Installation aus unbekannten Quellen" in den Sicherheitseinstellungen
   - Folgen Sie den Anweisungen auf dem Bildschirm

3. **Installation von Phyphox**
   - Öffnen Sie den Google Play Store
   - Suchen Sie nach "Phyphox"
   - Installieren Sie die App von "RWTH Aachen University"

### Phyphox-Einrichtung

1. Öffnen Sie die Phyphox-App
2. Tippen Sie auf "Fernzugriff erlauben"
3. Notieren Sie die angezeigte IP-Adresse (z.B. 192.168.1.5:8080)
4. Lassen Sie Phyphox im Hintergrund laufen

## Erste Schritte

### App starten

1. Tippen Sie auf das FractalSense EntaENGELment-Symbol auf Ihrem Startbildschirm
2. Beim ersten Start wird ein Einführungstutorial angezeigt
3. Folgen Sie den Anweisungen auf dem Bildschirm

### Verbindung mit Phyphox herstellen

1. Tippen Sie im Hauptmenü auf "Einstellungen"
2. Wählen Sie "Sensor-Verbindung"
3. Geben Sie die IP-Adresse von Phyphox ein
4. Tippen Sie auf "Verbinden"
5. Bei erfolgreicher Verbindung erscheint eine Bestätigung

### Benutzeroberfläche kennenlernen

Die Benutzeroberfläche besteht aus folgenden Hauptbereichen:

- **Visualisierungsbereich**: Zeigt das Fraktal und visuelle Effekte
- **Steuerleiste**: Am unteren Bildschirmrand für Hauptfunktionen
- **Seitenmenü**: Zugriff auf erweiterte Einstellungen (durch Wischen von links)
- **Modusauswahl**: Oben rechts für den Wechsel zwischen verschiedenen Modi

## Hauptfunktionen

### Fraktal-Visualisierung

1. **Grundlegende Navigation**
   - Zoomen: Spreizen oder Zusammenziehen mit zwei Fingern
   - Verschieben: Ziehen mit einem Finger
   - Zurücksetzen: Doppeltippen auf den Bildschirm

2. **Fraktaltypen**
   - Mandelbrot-Set (Standard)
   - Julia-Set (tippen Sie auf "Fraktaltyp" und wählen Sie "Julia")
   - Burning Ship (tippen Sie auf "Fraktaltyp" und wählen Sie "Burning Ship")

3. **Detaileinstellungen**
   - Iterationen: Bestimmt die Detailtiefe (höhere Werte = mehr Details, aber langsamere Berechnung)
   - Auflösung: Bestimmt die Bildqualität (höhere Werte = bessere Qualität, aber langsamere Berechnung)

### Sensor-Integration

1. **Aktivieren der Sensoren**
   - Tippen Sie auf "Sensor-Modus" in der Steuerleiste
   - Wählen Sie die gewünschten Sensoren (Beschleunigung, Gyroskop, etc.)

2. **Sensor-Steuerung**
   - Neigen nach vorne/hinten: Zoom ein/aus
   - Neigen nach links/rechts: Horizontale Bewegung
   - Drehen: Änderung der Parameter für Julia-Set

3. **Empfindlichkeit anpassen**
   - Tippen Sie auf "Einstellungen" > "Sensor-Empfindlichkeit"
   - Passen Sie die Schieberegler für jeden Sensor an

### ResonanceEnhancer

1. **Klangmodi**
   - Tippen Sie auf "Klang" in der Steuerleiste
   - Wählen Sie einen Modus:
     - Harmonisch: Basierend auf dem goldenen Schnitt
     - Fraktal: Klänge mit fraktaler Struktur
     - Resonant: Basierend auf Resonanzfrequenzen
     - Spektral: Basierend auf dem Frequenzspektrum

2. **Klangeinstellungen**
   - Lautstärke: Schieberegler unten im Klangmenü
   - Basisfrequenz: Bestimmt die Grundtonhöhe
   - Komplexität: Bestimmt die Anzahl der Obertöne

3. **Farbmodi**
   - Tippen Sie auf "Farbe" in der Steuerleiste
   - Wählen Sie ein Farbschema:
     - Resonant: Violett zu Gold
     - Harmonisch: Basierend auf Fibonacci-Sequenz
     - Spektral: Basierend auf dem Lichtspektrum
     - Fraktal: Basierend auf Mandelbrot-Iteration
     - Kosmisch: Sternenhimmel mit Nebeln

### Hypergraph-Visualisierung

1. **Aktivieren des Hypergraphen**
   - Tippen Sie auf "Hypergraph" in der Steuerleiste
   - Wählen Sie die Datenquellen für den Graphen

2. **Interaktion mit dem Graphen**
   - Tippen auf Knoten: Zeigt Details zu diesem Datenpunkt
   - Ziehen von Knoten: Ändert die Graphstruktur
   - Spreizen/Zusammenziehen: Ändert die Ansicht des Graphen

## Präsentationsmodus

Der Präsentationsmodus ist speziell für Vorführungen vor Publikum konzipiert.

### Aktivieren des Präsentationsmodus

1. Tippen Sie auf "Menü" > "Präsentationsmodus"
2. Wählen Sie eine der vordefinierten Präsentationen oder erstellen Sie eine eigene

### Vordefinierte Präsentationen

1. **Einführung in Fraktale** (3 Minuten)
   - Grundlegende Fraktalvisualisierung
   - Zoom-Demonstration
   - Einfache Erklärungen

2. **Multisensorische Erfahrung** (5 Minuten)
   - Fraktalvisualisierung mit Klang und Farbe
   - Demonstration verschiedener Klang- und Farbmodi
   - Interaktive Elemente

3. **EntaENGELment-Konzept** (7 Minuten)
   - Vollständige Demonstration aller Funktionen
   - Erklärung der philosophischen Konzepte
   - Praktische Anwendungsbeispiele

### Eigene Präsentation erstellen

1. Tippen Sie auf "Neue Präsentation"
2. Wählen Sie die gewünschten Module und deren Reihenfolge
3. Stellen Sie die Dauer für jeden Abschnitt ein
4. Speichern Sie die Präsentation mit einem Namen

### Präsentation durchführen

1. Wählen Sie die gewünschte Präsentation
2. Tippen Sie auf "Starten"
3. Die App führt Sie durch die Präsentation mit Hinweisen
4. Tippen Sie auf "Weiter", um zum nächsten Abschnitt zu gelangen
5. Tippen Sie auf "Pause", um die Präsentation anzuhalten

## Fehlerbehebung

### Verbindungsprobleme

**Problem**: App kann keine Verbindung zu Phyphox herstellen

**Lösungen**:
1. Stellen Sie sicher, dass Phyphox läuft und "Fernzugriff erlauben" aktiviert ist
2. Prüfen Sie, ob beide Geräte im selben WLAN-Netzwerk sind
3. Überprüfen Sie die eingegebene IP-Adresse
4. Starten Sie beide Apps neu

### Leistungsprobleme

**Problem**: App reagiert langsam oder ruckelt

**Lösungen**:
1. Reduzieren Sie die Fraktal-Auflösung in den Einstellungen
2. Verringern Sie die Anzahl der Iterationen
3. Deaktivieren Sie nicht benötigte Sensoren
4. Schließen Sie andere Apps im Hintergrund

### Klangprobleme

**Problem**: Kein Ton oder verzerrter Ton

**Lösungen**:
1. Prüfen Sie die Lautstärkeeinstellungen des Geräts
2. Stellen Sie sicher, dass der Stummschalter nicht aktiviert ist
3. Reduzieren Sie die Komplexität der Klänge in den Einstellungen
4. Starten Sie die App neu

### App-Abstürze

**Problem**: App stürzt ab oder friert ein

**Lösungen**:
1. Starten Sie die App neu
2. Prüfen Sie, ob genügend freier Speicherplatz vorhanden ist
3. Aktualisieren Sie die App auf die neueste Version
4. Setzen Sie die App-Einstellungen zurück (Menü > Einstellungen > Zurücksetzen)

## Glossar

- **Fraktal**: Eine mathematische Struktur mit selbstähnlichen Mustern auf verschiedenen Skalen
- **Mandelbrot-Set**: Ein bestimmtes Fraktal, benannt nach Benoît Mandelbrot
- **Julia-Set**: Eine Familie von Fraktalen, die mit dem Mandelbrot-Set verwandt sind
- **Iteration**: Wiederholte Anwendung einer mathematischen Funktion
- **Selbstähnlichkeit**: Eigenschaft, bei der ein Teil dem Ganzen ähnelt
- **Resonanz**: Schwingungsphänomen, bei dem ein System bei bestimmten Frequenzen besonders stark reagiert
- **Hypergraph**: Erweiterung eines Graphen, bei der Kanten mehr als zwei Knoten verbinden können
- **Autopoiesis**: Selbsterschaffung und -erhaltung eines Systems
- **Mereotopologie**: Theorie über die Beziehung zwischen Teil und Ganzem
- **EntaENGELment**: Philosophisches Framework, das Technologie, Bewusstsein und Natur verbindet
